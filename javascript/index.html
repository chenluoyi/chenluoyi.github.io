#
gulp入门引导

一、gulp是什么?

  Gulp 是基于node.js的一个前端自动化构建工具，开发这可以使用它构建自动化工作流程（前端集成开发环境）。
  使用gulp你可以简化工作量，让你把重点放在功能的开发上，从而提高你的开发效率和工作质量。
  例如：你可以用gulp可以网页自动刷新，和MVVM开发模式很相似，如果你对vue.js有所了解的话，那么你一定不会陌生。你也可以使用gulp对sass进行预处理、代码检测、图片优化压缩、只需要一个简单的指令就能全部完成。

二、安装gulp

 npm install gulp -g  //全局安装gulp,注意是全局安装

三、初始化gulp

npm install gulp //新建项目文件夹后，在当前项目文件初始化gulp项目

四、引入pack.json包

npm init //在当前项目生成pack.json包,建议养成习惯建立起属于自己的pack.json包

五、使用实例

一、 首先在项目文件下面新建gulpfile.js文件
二、在gulpfile.js文件种引入gulp模块:
    var gulp = require('gulp')
三、创建默认任务
    gulp.task('hello',function () {
        console.log('hello wolrd')
    })
四、测试
    在node中输入: gulp hello //hello wolrd为执行名为hello的task的结果
以上就是gulp入门引导，请自己动手操作
